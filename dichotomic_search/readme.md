# Dichotomic Search

### The Problem

Find a value inside an ordered array.

### Input >> Desired Output / Test Cases

['a','b','d','e','g','h','j','l','m','q','x'] 'z' >> null

['a','b','d','e','g','h','j','l','m','q','x'] 'e' >> 'e'

[1,2,3,5,6,10,15] 20 >> null

[1,2,3,5,6,10,15] 4 >> null

[1,2,3,5,6,10,15] 3 >> 3
